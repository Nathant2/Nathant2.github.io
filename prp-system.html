About the PRP Management System

The PRP Management System is a full-stack web application designed for the UNC-Chapel Hill Computer Science department to help manage the Preliminary Research Paper (PRP) process for PhD students. The system streamlines scheduling, tracking student forms and data, and creating announcements, providing a centralized platform for both students and faculty.

This project was developed using MongoDB, JavaScript, React and Node.js, focusing on data management, secure user access, and ease of use for administrative staff and faculty members.
